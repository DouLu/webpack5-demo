- @babel/preset-env

## 会用到的 Babel plugin

- @babel/plugin-transform-runtime

## Babel polyfill

polyfill 是为环境提供不支持特性的一类文件或库，既有 Babel 官方提供的库，也有第三方提供的。
即：core-js 和 regenerator-runtime
